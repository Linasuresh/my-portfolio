What I do

Below is a showcase of my projects and case studies, where I have applied my skills in cloud computing, DevOps, application development, and scripting to solve real-world challenges. Each project reflects my expertise in designing scalable solutions, leveraging emerging technologies, and optimizing system performance.

Green Wardrobe

Green Wardrobe is a sustainable clothing website that enables users to buy and sell unused clothes at reasonable rates, promoting eco-friendly fashion and reducing textile waste. I developed this platform using Spring Suite for backend services, Python and JavaScript for dynamic functionalities, and Git for version control, ensuring seamless collaboration and code management. Through this project, I gained hands-on experience in full-stack development, database integration, and building scalable, user-friendly web applications.

Decentralized Chat Application

I developed a decentralized chat application powered by blockchain technology, where users can communicate securely while transactions run on Ether expenses. The application was deployed on the Ethereum blockchain using the Sepolia testnet, ensuring decentralized and tamper-proof messaging. I leveraged HTML, Python, JavaScript, and smart contract development to build this solution, gaining hands-on experience in blockchain integration, Web3 technologies, and decentralized application (dApp) development.

Automated infrastruture using CloudFormation

In this project, I automated infrastructure deployment using AWS CloudFormation, where I created and managed stacks to streamline the setup of resources in the cloud. I utilized Infrastructure Composer to deploy these stacks, simplifying the process of infrastructure management and ensuring repeatable, scalable deployments. This project gave me hands-on experience with Infrastructure as Code (IaC), enabling efficient automation and configuration of cloud resources.

Serverless architecture on AWS

In this project, I developed an inventory tracking system where stores upload inventory files to Amazon S3, triggering a series of automated events. A Lambda function reads the inventory file and inserts the data into a DynamoDB table, while another Lambda function monitors inventory levels and sends an SNS notification when stock is low. Using Amazon Cognito for authentication, a serverless, web-based dashboard provides real-time updates of inventory levels, ensuring efficient monitoring and prompt restocking notifications via SMS or email.

Azure Intersite connectivity

In this project, I configured Global Virtual Network Peering to enable secure and seamless communication between Azure virtual networks across different regions. Additionally, I deployed Blob Storage with private access, ensuring that the storage is securely accessible only within the virtual network, enhancing data security and network isolation.

Azure Dynamic Web Hosting

I developed and hosted a dynamic website on Azure, leveraging technologies such as Azure App Services, HTML, CSS, JavaScript, and a SQL database for dynamic content management. The website is designed to handle user interactions, store data securely, and scale efficiently with Azure’s cloud services, ensuring high availability and performance.
